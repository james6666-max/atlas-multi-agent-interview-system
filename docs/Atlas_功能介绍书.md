# Atlas · 多智能体 AI 面试助手 —— 功能介绍书

> 本文系统梳理 Atlas **目前已实现**的全部功能,供路演 / PPT 取材。
> 一句话定位:本地优先的多 Agent 实时面试助手,覆盖「面试前准备 → 面试中辅助 → 面试后复盘」全链路,中英双语。
> 工程基线:后端 114 单元测试通过 · 50 题中英评测 250/250 · 可一键打包为 Windows 安装包(内置后端、开箱即用)。

---

## 0. 总体架构(一句话讲清)

```
Electron + React 前端(三幕界面 / 流式渲染 / 中英 i18n)
        │ REST + SSE(127.0.0.1:8000)
        ▼
FastAPI 后端(orchestrator_v0:app)
  ├─ 共享黑板事件总线 + 编排器(链式分发,可观测)
  ├─ Agents:感知 → 简历 + 检索 → 技术/行为 → 评审
  ├─ 分级 LLM 路由:本地 Ollama + 云端 OpenAI 兼容(流式 + 脱敏)
  ├─ 陪练:简历/JD/公司职位 驱动出题 + 评分 + 复盘
  └─ 多模态:RapidOCR(截图)· faster-whisper(语音)
```

核心创新:把面试拆成**多个专业 Agent 协同** + **「生成—评审」分离**,突破单 Agent「一个 Prompt 同时做五件事」的天花板。

---

## 1. 多 Agent 后端架构

事件驱动:`/ask → 感知 → 简历 + 检索 → 技术/行为 → 评审 → 答案`,Agent 间通过共享黑板异步通信,新增 Agent 不影响现有模块。

| Agent | 职责 | 关键点 |
|---|---|---|
| **Perception 感知** | 判断输入是否完整面试问题 + 意图分类 | 过滤闲聊/半句话;分类:技术/算法/系统设计/行为/简历追问/忽略 |
| **Resume 简历画像** | 读取简历/JD/知识库,回灌相关上下文 | 关键词片段匹配,给回答 Agent 提供素材 |
| **RAG 检索增强** | 本地关键词检索(知识库 + 会话回放) | 中文 **bigram** 切分,显著提升中文召回;不联网、无向量库 |
| **Tech / Behavioral** | 技术/算法/系统设计 走 Tech;行为/简历追问 走 Behavioral | 行为面强调 STAR、不编造简历事实 |
| **Critic 评审** | 标准化最终审核,产出最终答案 | 规则引擎(非 LLM):AI 腔 / 隐私泄露 / 长度 / 简历事实 / 技术质量;不通过给安全降级答案 |

- **真实 trace**:`/trace/{session_id}` 直接 dump 事件流,前端 Agent 链路是**真实事件**而非推断。
- 可观测、可回放,便于调试与演示。

## 2. 分级 LLM 路由 + 流式 + 隐私守护

- **本地 Ollama**:零成本、最私密(如 qwen2.5:7b)。
- **云端 OpenAI 兼容**:Groq / DeepSeek / 通义千问等,一套客户端通吃。
- **Hybrid 混合**:云端优先(低延迟流式),失败自动**回落本地**。
- **SSE 流式输出**:边生成边显示,显示首字 / 总耗时、provider 徽章。
- **一键「测试连接」**:真打一次模型,明确告诉你当前走的是 cloud / ollama / 是否回落 / 延迟。
- **出站隐私脱敏**:发往云端前,对邮箱 / 手机号 / 身份证 / API Key 等做确定性脱敏(非 LLM,可审计);本地调用不脱敏(数据不出本机)。
- **热重载**:在设置里改模型/Key 立即生效,无需重启。

## 3. 面试前 · 准备(Prep)

### 3.1 候选人资料
- 简历 / JD / 知识库:可**直接粘贴**或**上传文件**。
- **文件解析**:支持 **PDF / TXT / DOCX**,后端自动抽取纯文本(PDF 走 pypdf)。
- 目标**公司 / 职位 / 重点方向**:其中公司、职位、简历为**必填**(带校验与红框提示)。
- 资料持久化到本机;**结合简历 + JD + 公司职位综合定制出题**。

### 3.2 模拟面试陪练
- 选择题数(3 / 5 / 7),一键开始。
- 出题来源:简历/JD/公司职位驱动(有 LLM 时用 LLM 生成,否则确定性题库;**中英双语题目**)。
- 逐题**规则化评分** + 改进建议。
- **自适应追问**:回答较弱或行为/简历类问题时,自动追问(有上限,不发散)。
- 结束生成**复盘报告**:总分、亮点、待改进、推荐练习、逐题回顾、按题型统计。

## 4. 面试中 · 实战(Live)

| 入口 | 流程 |
|---|---|
| **文字提问** | 流式逐字答案 + Critic 评审 + 上下文/检索来源 + 真实 Agent 链路 |
| **截图问答** | 对屏幕题目截图 → 本地 OCR(RapidOCR)→ 答案(适合算法/系统设计/代码题) |
| **语音提问** | 录音 → 本地 Whisper(faster-whisper)转写 → 答案 |

- **全局热键**(应用未聚焦也生效):
  - `Ctrl+Shift+A` 截图问答 · `Ctrl+Shift+V` 剪贴板题目流式作答 · `Ctrl+B` 显示/隐藏 · `Ctrl+Q` 退出 · `Ctrl+[ /]` 透明度 · `Ctrl+方向键` 移动窗口。
- **隐形悬浮**:窗口对 Zoom / Teams / 腾讯会议**屏幕共享不可见**(内容保护)。
- **Markdown 渲染**:答案的标题、列表、代码块、表格等正常排版。

## 5. 面试后 · 复盘(Review)

- 历史记录:逐题的类型、Agent、评分、问题点回看。
- 一键**生成复盘报告**;**导出 Markdown**;支持**清空历史 / 重置会话**。
- 报告维度:综合得分、亮点、待改进、推荐练习。

## 6. 横切能力

### 6.1 中 / EN 全局双语
- 右上角一键切换:**界面文案 + 模拟面试题目 + 回答语言**同步切换。
- 切换语言同时作为后端 hint,提升 Whisper 语音识别准确率。

### 6.2 多模态输入(全本地)
- **截图 OCR**:RapidOCR 本地识别,无需联网。
- **语音 STT**:faster-whisper 本地转写(首次自动下载模型)。
- **文字**:直接输入或剪贴板。

### 6.3 数据与隐私(本地优先)
- 用户数据(简历/JD/资料/问答记录)默认**本机存储**。
- 仅在主动配置云端模型时联网;联网前脱敏。
- API Key 仅存本机,不回显、不上传到所配置服务商之外。

### 6.4 体验与健壮性
- 顶层 **ErrorBoundary**:组件出错显示提示而非白屏。
- 三幕清晰的导航(准备 / 实战 / 复盘),不再把功能堆在一页。
- 流式光标、provider 徽章、首字/总耗时等细节反馈。

## 7. 工程化与可信度

- **测试**:后端 114 单元测试;`check_phase2` 一键串起 py_compile + 单测 + 冒烟 + 评测。
- **评测集(项目的“方向盘”)**:50 题中英混合,覆盖技术/算法/系统设计/行为/简历追问/**陷阱题**六类,当前 **250/250**;每次改动都有量化反馈。
  - 陷阱题理念:测「该不该响应」——闲聊不答、半句话不答、隐私不回灌、没有的经历不编造。
- **打包分发**:PyInstaller 把后端冻结成 sidecar,electron-builder 出 **NSIS 安装包**;普通用户**双击安装、开箱即用**(无需自己装 Python)。
- **可观测**:真实事件 trace + 后端日志,问题可定位。
